### [Redwood Finance Automation -- Help Site Redesign](https://maughtycat.github.io/portfolio/projects/redwood.html)

The complete case study of how I rebuilt a broken enterprise documentation portal from the ground up. A proof-of-concept workflow showing how AI agents can automate the entire content lifecycle, from code change to published content update. The workflow pattern has been validated in production at Redwood Software, where FlareGuard automated content quality scoring and eliminated customer escalations to zero.

## Who I Am

I'm a technical writer who engineers systems, not just documents them. Seven years of experience building documentation for enterprise software, hardware products, and IoT platforms -- and for the last year, building the AI systems that generate content itself.

I work in Markdown, Git, and TypeScript daily. I use AI coding agents (Claude Code, Codex, Cursor, Cline, Hermes Agent, Kilo) as force multipliers for both writing and engineering tasks. I believe docs-as-code is the only sustainable approach to technical content at scale, and I believe prompt engineering is the new technical writing.

**What sets me apart:** I had never used agentic AI before December 2025. Within six months, I had redesigned an enterprise help site, built a PDF pipeline, created interactive product simulations, and restructured a release notes format -- all powered by AI coding agents that were new to me months earlier. I learn, and I learn fast.

## Tech Stack

This portfolio is intentionally simple -- static HTML and CSS, no JavaScript framework, no build step. It loads fast, works on every browser, and is fully accessible. The content is the product.

- HTML5 / CSS3 (dark/light mode, responsive, print-optimized)
- No dependencies, no npm, no bundler
- Hosted on GitHub Pages
